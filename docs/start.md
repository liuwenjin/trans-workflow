# 快速开始

欢迎来到 **Trans-Workflow**。

本教程将通过一个预设的示例工作流，带你快速完成从**执行**到**结果导出**的全过程。

* **零门槛**：无需安装软件，无需配置数据库。
* **零基础**：无需掌握复杂的工作流引擎逻辑。
* **即插即用**：纯浏览器环境下的高效体验。

---

## 第一步：进入编辑器界面

访问在线编辑器：[transweb.cn/workflow](https://transweb.cn/workflow)

![编辑器界面](https://transweb.cn/workflow/docs/images/start001.png "编辑器界面")

进入页面后，你会看到一个布局直观的集成环境。为了方便后续操作，建议先了解界面布局：

* **Header（顶栏）**：左侧可自定义**工作流标题**；右侧为**全局工具栏**，包含新增可选 Agent 按钮、修改 Agent 列表菜单、Agent 导航菜单及工作流导出按钮。
* **Config Area（配置区）**：第二行区域，用于定义工作流的入口 Agent 及后续节点的关联 ID。
* **Preview & Switcher（状态与切换）**：第三行区域，实时展示工作流节点的**运行状态**，同时支持点击切换查看不同节点的详情。
* **Runtime Console（运行控制台）**：第四行区域，即当前选中 Agent 的**交互界面**，也是执行任务的核心地带。

---

## 第二步：一键执行工作流

![运行控制台](https://transweb.cn/workflow/docs/images/start002.png "运行控制台")

在页面底部的 **“运行控制台”** 区域，你只需完成以下操作：

1. **输入指令**：在文本框内输入你的需求描述。

2. **触发任务**：点击 **“立即生成”**。
> *注意：首次使用可能需要进行手机验证码登录。*
> 

![开始运行后](https://transweb.cn/workflow/docs/images/start003.png "开始运行后")


3. **监控进度**：观察第三行的**步骤条**。随着各个 Agent 节点依次被点亮，代表工作流正在按顺序流转。

![运行完成](https://transweb.cn/workflow/docs/images/start004.png "运行完成")

当步骤条全部变为完成状态，恭喜你，工作流已成功执行完毕！

---

## 第三步：结果查阅与本地存档

工作流运行结束后，最终产物会直接呈现在控制台的显眼位置。

### 1. 溯源中间结果

Trans-Workflow 支持**全链路回溯**。你可以点击第三行步骤条中的任意**历史节点**，查看该步骤生成的中间数据。

![溯源中间结果](https://transweb.cn/workflow/docs/images/start006.png "溯源中间结果")


### 2. 导出成果物

如果你对生成结果满意，可以将其导出为独立的网页文件：

* 在结果展示区点击 **“操作”** -> 选择 **“存档 HTML”**。
* 系统将自动下载一个包含结果内容的 HTML 文件，方便你离线查看或分享。

![导出成果物](https://transweb.cn/workflow/docs/images/start005.png "导出成果物")
