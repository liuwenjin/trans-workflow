🔗

多 Agent 同步串联

支持多个 AI Agent 节点流水线式无缝交接，前一个 Agent 的输出（DataId / InputItem）自动作为下一个 Agent 的初始输入，构建复杂协同智能体。

👁️

执行过程透明 · 可干预

工作流执行中每个 Agent 节点的过程及结果全程可见，可随时中断后继续，每个节点的结果内容支持 AI 润色或手动调优，掌控每一环节。

🔄

节点级回溯与重触发

支持在任意中间节点一键「重新触发」或「状态重置」，无需从头运行，极大提升调试与交互效率，迭代更自由。

🎛️

动态自由编排

界面上动态新增、删除、调换 Agent 节点，随时重新生成步骤流。编排完成后可导出新版工作流 HTML 文件，方便分享、归档或二次部署。

💾

自动持久化运行结果

执行过程数据及最终产出自动存储至个人 transweb Agent 基座后台数据库，随时登录查看、编辑或导出，数据所有权完全归属使用者。

📦

零依赖 · 单文件交付

一个 HTML 文件包含编辑器 + 工作流引擎，无需数据库、无需后端。复制到任意 Web 服务器即可运行，私有化部署秒级完成。
